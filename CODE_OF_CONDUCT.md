# Code of Conduct

## Our standard

This project expects respectful, direct, and technically grounded collaboration.

Examples of expected behavior:

- discuss ideas and tradeoffs in good faith
- provide reproducible bug reports and clear technical context
- challenge design choices without attacking people
- keep feedback focused on the code, product, and user impact

Examples of unacceptable behavior:

- personal attacks or harassment
- hostile or dismissive language
- deliberate bad-faith disruption
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, issues, pull requests, or other contributions that violate this code of conduct.

## Reporting

If you see behavior that should be reviewed, open a private security-style report through the contact path defined in [SECURITY.md](./SECURITY.md).
